# RWB Password Manager

RWB Password Manager is a lightweight Python application designed to help users take back control of their passwords.

In today's digital world many people depend completely on cloud-based password managers. While convenient, this also means that access to important credentials depends on external services and a working internet connection.

The idea behind RWB Password Manager is simple: **your passwords should remain under your own control.**

## Why this project exists

If a cloud service disappears, an account becomes locked, or the internet is unavailable, access to important credentials could be lost.

In my personal case I manage more than **150 credentials**, including:

- usernames  
- passwords  
- PIN codes  
- web addresses  
- other sensitive information  

Losing access to that data would be a serious problem.

RWB Password Manager was created as a local solution that keeps password data accessible without depending on external services.

## Features

- Local password vault
- User-controlled data storage
- Simple interface for managing credentials
- Create, search and edit entries
- Backup functionality
- Designed for simplicity and control
- Android Mobile

## Philosophy

Technology should help users **gain control over their own data**, not lose it.

This project reflects that idea by keeping password data locally available and independent of external cloud services.
